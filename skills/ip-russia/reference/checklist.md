# IP Checklist

Full checklist across all IP object types. Use for `checklist` command or comprehensive IP audit.

## Pre-Creation

- [ ] Identify what you're creating (product, content, brand, technology)
- [ ] Classify each creation using the classification tree in SKILL.md
- [ ] Check Art. 1225 — is it on the list? If not → no IP protection
- [ ] Check existing rights before launching (trademarks, patents, copyright)
- [ ] NDAs in place for employees and contractors working on IP

## Copyright

- [ ] Works fixed in tangible form
- [ ] Authorship documented (git, drafts, timestamps)
- [ ] Copyright notices on published works (optional but recommended)
- [ ] Employee-created works → check service work rules (Art. 1295)
- [ ] Commissioned works → check work-for-hire contract (Art. 1296)
- [ ] Open-source licenses reviewed and compliant
- [ ] Third-party content licensed or falls within free use

## Computer Programs & Databases

- [ ] Source code in version control with clear authorship
- [ ] Optional registration with Rospatent (evidence of creation)
- [ ] Database: document investment (sui generis right)
- [ ] Third-party libraries: license audit (GPL, MIT, Apache, etc.)
- [ ] CLA (Contributor License Agreement) for open-source projects

## Patents

- [ ] Prior art search done (rospatent.py search, Espacenet, Google Patents)
- [ ] No public disclosure before filing
- [ ] Application drafted (description, claims, drawings)
- [ ] Filed with Rospatent — priority date secured
- [ ] Substantive examination requested (within 3 years for inventions)
- [ ] Annual maintenance fees paid
- [ ] International filing considered (PCT within 12 months)

## Trademarks

- [ ] Trademark search conducted (rospatent.py search, TMview)
- [ ] Distinctiveness confirmed (not descriptive/generic)
- [ ] Application filed with Rospatent
- [ ] Registration certificate received
- [ ] Mark used in commerce (3-year non-use = cancellation risk)
- [ ] Renewal tracked (every 10 years)
- [ ] International registration considered (Madrid system)
- [ ] Monitoring for confusingly similar marks

## Means of Individualization

- [ ] Trade name in EGRUL (automatic upon incorporation)
- [ ] Commercial name used consistently at business location
- [ ] Geographical indications considered (if product quality linked to region)

## Know-How / Trade Secrets

- [ ] Secret information identified and documented
- [ ] Physical security measures implemented
- [ ] Digital security measures implemented
- [ ] NDAs signed by all employees and contractors
- [ ] Documents marked "confidential" / "commercial secret"
- [ ] Access restricted on need-to-know basis
- [ ] Departure procedures in place

## Contracts & Agreements

- [ ] All IP licenses in writing
- [ ] Patent/trademark licenses registered with Rospatent
- [ ] Assignment agreements registered (for registered IP)
- [ ] Remuneration specified (license without remuneration = void)
- [ ] Territory and scope clearly defined
- [ ] Moral rights understood (inalienable, even in assignment)

## Enforcement

- [ ] Monitoring for infringement (market watch, online search)
- [ ] Cease-and-desist template ready
- [ ] Evidence preservation procedures
- [ ] Legal counsel identified for IP litigation
- [ ] Statute of limitations noted (3 years from knowledge)

## International

- [ ] Berne Convention coverage verified (copyright auto-protected abroad)
- [ ] Paris Convention priority used (patents/trademarks within 6–12 months)
- [ ] Madrid system for international trademark registration
- [ ] PCT for international patent filing
- [ ] National treatment confirmed in target countries

## Quick Reference: Deadlines

Verify current deadlines in ConsultantPlus / ГАРАНТ.

| Action | Deadline | Missing = |
|---|---|---|
| Patent examination request | 3 years from filing | Application withdrawn |
| Patent annual fees | Year 3+ (inventions), year 1+ (utility models) | Patent lapses |
| Trademark renewal | Every 10 years | Registration cancelled |
| Trademark opposition | Within published opposition period | Right to oppose lost |
| Infringement claim | 3 years from knowledge | Time-barred |
| Convention priority (patent) | 12 months from first filing | Priority lost |
| Convention priority (trademark) | 6 months from first filing | Priority lost |
