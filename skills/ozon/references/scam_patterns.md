# Scam Patterns — Ozon Marketplace

Use this as a checklist when analyzing products. Each pattern is a red flag or warning sign.

## Category 1: Price Anomalies

| Pattern | Severity | Description |
|---------|----------|-------------|
| Price < 40% of market average | 🔴 Critical | Identical product for half the price — almost certainly scam |
| Price < 60% of market average | 🟡 Warning | Suspicious but possible during sales |
| No discount but price matches competitors | 🟢 Normal | Expected market behavior |
| "Too good to be true" bundle | 🔴 Critical | e.g., flagship phone + case + charger for price of phone alone |

### Market price reference points (2024-2025)
- Budget laptop (i3/8GB/256SSD): 25,000-40,000 ₽
- Mid-range laptop (i5/16GB/512SSD): 45,000-75,000 ₽
- Gaming laptop (RTX 4060): 70,000-120,000 ₽
- Flagship phone: 60,000-120,000 ₽
- Budget phone: 8,000-20,000 ₽

## Category 2: Seller Red Flags

| Pattern | Severity | Description |
|---------|----------|-------------|
| Account age < 3 months | 🔴 Critical | Especially for electronics > 10,000₽ |
| Rating < 3.5 | 🟡 Warning | Below average seller quality |
| No reviews on seller page | 🔴 Critical | New or abandoned seller |
| Name resembles official but isn't | 🔴 Critical | e.g., "ASUS_Official_Store2025" (fake) |
| Sells only 1-2 products | 🟡 Warning | Could be dropshipper or scammer |
| No response to negative reviews | 🟡 Warning | Indicates poor customer service |
| High sales but very young account | 🔴 Critical | Volume mismatch — suspicious activity |

## Category 3: Review Manipulation

| Pattern | Severity | Description |
|---------|----------|-------------|
| All reviews posted within 48 hours | 🔴 Critical | Classic review bombing |
| All reviews are 5★ with generic text | 🔴 Critical | Bot reviews ("Отличный товар!", "Рекомендую!") |
| Review text is nearly identical across reviews | 🔴 Critical | Copy-paste bot reviews |
| Reviewer accounts have no other activity | 🔴 Critical | Fake accounts |
| Sudden spike of reviews after quiet period | 🟡 Warning | Purchased reviews |
| Photos in reviews don't match the product | 🔴 Critical | Recycled from other listings |
| Reviews in wrong language for the platform | 🟡 Warning | Imported fake reviews |
| Rating 4.9+ but many detailed negative reviews | 🔴 Critical | Rating manipulation |

## Category 4: Product Spec Manipulation

| Pattern | Severity | Description |
|---------|----------|-------------|
| Vague CPU spec: "Intel i7" without model | 🔴 Critical | Could be 10-year-old i7 |
| "RTX 4060" at price of GT 1030 | 🔴 Critical | Bait-and-switch or fake specs |
| RAM listed as "up to X GB" (not actual) | 🟡 Warning | Misleading — actual RAM is lower |
| SSD listed as "1TB" with no brand/model | 🟡 Warning | Could be low-quality no-name drive |
| Screen resolution listed as "Full HD" for unclear size | 🟡 Warning | Verify actual resolution in specs |
| "Процессор: Современный" with no specifics | 🔴 Critical | Hiding weak hardware |
| Specs don't match manufacturer website | 🔴 Critical | Listing is fraudulent |
| Chinese no-name brand with flagship specs at budget price | 🔴 Critical | Specifications are fabricated |

## Category 5: Listing Quality

| Pattern | Severity | Description |
|---------|----------|-------------|
| Stock/placeholder photos only | 🟡 Warning | Seller doesn't have actual product |
| Title contains excessive keywords/emoji | 🟡 Warning | SEO-stuffed, low quality seller |
| Description is copy-pasted from AliExpress | 🔴 Critical | Reseller or scammer |
| Multiple "identical" listings from same seller at different prices | 🔴 Critical | Price manipulation scheme |
| "Премиум" or "Original" in title but no brand verification | 🟡 Warning | Potentially counterfeit |

## Category 6: Delivery & Return

| Pattern | Severity | Description |
|---------|----------|-------------|
| No return policy stated | 🟡 Warning | Seller avoiding responsibility |
| Delivery time > 30 days for domestic product | 🟡 Warning | May be shipping from China despite "Russian" listing |
| Ships from "other city" with no specifics | 🟡 Warning | Dropshipping from unknown source |
| Cash-on-delivery only, no Ozon delivery | 🟡 Warning | Bypassing Ozon's buyer protection |

## How to Use This Checklist

When analyzing a product, run through these categories systematically:

1. **Price**: Is the price realistic for the claimed specifications?
2. **Seller**: Check account age, rating, product count, response rate
3. **Reviews**: Look for bot patterns, timing anomalies, text similarity
4. **Specs**: Verify claimed specs against known hardware benchmarks
5. **Listing**: Check photo quality, description authenticity
6. **Delivery**: Verify shipping terms and return policy

**Scoring**:
- Each 🔴 Critical flag: -3 points (starting from 7)
- Each 🟡 Warning flag: -1 point
- Official store: +2 points
- Verified brand: +1 point

**Verdict**:
- 7-10: ✅ Safe to buy
- 4-6: ⚠️ Investigate further
- 0-3: 🔴 High scam risk
