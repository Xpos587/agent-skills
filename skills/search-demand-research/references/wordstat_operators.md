# Wordstat Operators for Research

Quick reference for Yandex Wordstat search operators. Use for precise research queries, not general collection.

## Operators

| Operator | Syntax | Effect | Research use |
|---|---|---|---|
| None | `—Д–Є–≥—Г—А–љ–Њ–µ –Ї–∞—В–∞–љ–Є–µ` | All word forms and tails | Default вАФ landscape discovery |
| Quotation | `"—Д–Є–≥—Г—А–љ–Њ–µ –Ї–∞—В–∞–љ–Є–µ"` | Only these words in any order | Exact demand for specific phrase |
| Exclamation | `!–Ї—Г–њ–Є—В—М` | Fixed word form | Check exact form frequency |
| Plus | `+—Б—В–∞–ї—М` | Includes stop-word in query | When stop-words matter |
| Brackets | `[–Ї—А–∞—Б–љ—Л–є –Љ—П—З]` | Fixed word order | Precise intent matching |
| Minus | `–Њ–±—Г–≤—М -–і–µ—В—Б–Ї–∞—П` | Excludes word | Filter noise from results |
| Grouping | `(–Ї—Г–њ–Є—В—М|–Ј–∞–Ї–∞–Ј–∞—В—М) X` | OR logic | Cover synonyms in one query |

## Research Patterns

| Goal | Operator pattern | Example |
|---|---|---|
| Pain demand | `"–Ї–∞–Ї —Г–ї—Г—З—И–Є—В—М" X` | `"–Ї–∞–Ї —Г–ї—Г—З—И–Є—В—М" –њ—А—Л–ґ–Њ–Ї` |
| Exact product search | `"X –Є–љ—Б—В—А—Г–Љ–µ–љ—В"` | `"–∞–љ–∞–ї–Є–Ј –≤–Є–і–µ–Њ" –Є–љ—Б—В—А—Г–Љ–µ–љ—В` |
| Commercial intent | `(–Ї—Г–њ–Є—В—М\|–Ј–∞–Ї–∞–Ј–∞—В—М\|—Ж–µ–љ–∞) X` | `(–Ї—Г–њ–Є—В—М\|—Ж–µ–љ–∞) —В—А–µ–љ–∞–ґ–µ—А` |
| Exclude DIY/info | `X -—Б–≤–Њ–Є–Љ–Є -–±–µ—Б–њ–ї–∞—В–љ–Њ -—Б–Ї–∞—З–∞—В—М` | `—А–µ–Љ–Њ–љ—В -—Б–≤–Њ–Є–Љ–Є -–±–µ—Б–њ–ї–∞—В–љ–Њ` |
| Competitor check | `"X vs Y"` | `"kinovea" —Д–Є–≥—Г—А–љ–Њ–µ` |
| Parent segment | `X —А–µ–±—С–љ–Њ–Ї\|–і–µ—В—Б–Ї–Є–є` | `—Д–Є–≥—Г—А–љ–Њ–µ –Ї–∞—В–∞–љ–Є–µ —А–µ–±—С–љ–Њ–Ї` |

## Rules

- Use operators deliberately for targeted research, not randomly during general collection
- Operator queries return smaller result sets вАФ interpret carefully
- Quotation marks restrict results significantly; use for validation of specific phrases, not discovery
- Minus operators help separate signal from noise in high-volume root masks
